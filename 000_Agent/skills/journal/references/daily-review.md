# Daily Review Reference

Use this reference when the user wants a reflective daily review, not only a work log.

## Reflection Lenses

Use one or two lenses only; do not turn daily review into a long interview.

| Lens | Prompt | Source Cards |
|---|---|---|
| Emotion root cause | 這個情緒底下，有什麼「期待」沒被滿足？或有什麼「恐懼」被觸動了？ | `AN1A2D`, `DU1`, `DU` |
| Body awareness | 這個感受在你身體哪裡？是緊繃、沉重，還是什麼感覺？ | `F6A2A1A5` |
| Past vs. present trigger | 這個反應是當下的情況如此，還是被過去某個記憶觸發的？ | `DU`, `DU1` |
| Behavior pattern | 今天有沒有「自動導航」的慣性反應？ | `DU1`, `A1A1A1A1A` |
| Ideal self gap | 你正在練習的方向，今天有進展還是退步？ | `A1D1A2A12`, `H2B` |
| Listening quality | 今天有沒有真正聽到某人的感受，而不只是話語？ | `A1D1A4`, `A1D1A2A13` |
| Small success | 今天有沒有一件很小但值得保留的成功？ | `A1A1G1A2A`, `A1A1G1A` |
| Micro stress | 今天有沒有什麼在無形中悄悄累積的壓力？ | `B1B1A2F1`, `B1B1A2F3` |
| Meaning moment | 今天哪件事讓你感到最有意義？ | `AW1C1`, `B1B1A3E` |
| Learning extraction | 今天學到的東西，在什麼情境可以提取或練習？ | `DZ1A1` |

## Question Patterns

- Emotion type + intensity: 「今天有沒有一個情緒值得記下來？」→ 展開完整情緒輪（必須完整展開，不可省略）
- Root cause: 「這個情緒底下，有什麼『期待』沒有被滿足？」
- Body: 「這個感受在你身體哪裡？」
- Pattern: 「這是單次事件，還是最近反覆出現的模式？」
- Ideal self: 「你正在練習的哪個方向，今天有用到嗎？」
- Small success: 「今天有沒有一件小事，讓你覺得自己多了一點什麼？」
- Meaning: 「今天哪個瞬間讓你感到最有意義或滿足？」

## Writing Rules

- Keep the output concrete and searchable.
- Preserve the user's own words for emotions, people, projects, and events.
- Do not diagnose the user's mental state.
- Emotion content: describe and label only, no personality evaluation.
- If the user is tired or vague, ask for one concrete event first.
- Do NOT generate tomorrow's task list.

## Card-Based Action Suggestions

After filing the journal entry, always run Step 3 from SKILL.md:
read the relevant cards from `400_Knowledge/study/01_processed` based on today's themes,
and produce 1-3 concrete action suggestions grounded in the card content.
