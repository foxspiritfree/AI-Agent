# Table And Style Reference

Use this reference when reviewing tables, reports, formal documents, visual style direction, campaign pages, posters, brand screens, or mood-specific layouts.

## Table Design

Classic table design should feel trustworthy, stable, and professional. It fits data reports, business documents, and formal information display.

- Use neutral colors or one restrained color family.
- Reduce borders; use whitespace to separate information.
- When borders are needed, use thin lines.
- Prefer subtle brightness differences over heavy color blocks.
- Use low-contrast header shading or a gentle gradient only when it clarifies hierarchy.
- Reduce internal contrast so data remains readable without visual noise.

## Style Presets

| Style | Typography | Color | Lines / Shape | Best Use |
|---|---|---|---|---|
| Classic | Neutral sans or restrained type | Black, white, gray, or same-color family | Thin lines, few borders | Reports, business documents, formal UI |
| Lively | Bold sans, heavy weight | High chroma, multiple hues with consistent tone | Slants, varying line weight, varied lengths | Campaigns, games, energetic announcements |
| Elegant | Thin Ming/serif, English serif | 2-4 low-chroma related colors | Fine, regular, low movement | Fashion, premium service, cultural events |
| Strong | Bold sans, heavy weight | Black/gray plus dark blue or dark brown | Thick lines, high contrast | Professional, forceful, masculine-coded contexts |

## Lively Style

- Use bold sans typography.
- Tilt groups of text consistently when creating speed or direction.
- Create rhythm through line thickness, angle, length, and perspective.
- Use high-chroma colors, but keep tone and saturation consistent.
- Avoid mixing muddy low-chroma colors with vivid colors.

## Elegant Style

- Use thin Ming or serif type when the context supports refinement.
- Keep variation low: fewer decorations, cleaner alignment, and calmer rhythm.
- Use a small color set: one main color, one support color, plus grayscale.
- Use warm, darker, or slightly yellowish colors such as beige, tea, brown, gray-purple, or wine red as accents.

## Strong Style

- Use heavy sans typography and high contrast.
- Favor black, white, gray, dark blue, steel blue, dark gray-blue, dark brown, wine brown, or gray pink-brown.
- For photos, increase contrast and reduce saturation to create a heavier, more forceful impression.

## Style Review Questions

- Does the selected style match the intended audience and trust signal?
- Are typography, color, line, and photo treatment pointing to the same mood?
- Is the style helping comprehension or only adding surface personality?
- Is the palette consistent in tone and saturation?
