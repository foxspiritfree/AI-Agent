# Visual Layout Reference

Use this reference when reviewing layouts, slides, dashboards, reports, cards, image/text composition, or visual hierarchy.

## Fast Understanding

- Treat quick comprehension as the main quality metric for information design.
- Confirm the purpose and priority before judging style.
- Reduce visual decisions the viewer must make: what to read first, what belongs together, and what action follows.

Source cards: `A4E4E5`, `A4P1A`.

## Alignment And Grouping

- Use grid, baseline, and object center of gravity to make alignment feel intentional.
- Group related information by proximity and spacing before adding borders.
- Use Gestalt grouping to make sections understandable at a glance.

Source cards: `A4E4E5A`, `A4E4E5B`.

## Density And Atmosphere

| Layout Density | Best For | Impression |
|---|---|---|
| High | Information-heavy, active, energetic screens | Busy, rich, lively |
| Low | Premium, calm, minimal communication | Spacious, refined, focused |

Source card: `A4E4E3`.

## Color And Readability

- Use brightness contrast for readability before relying on hue.
- Use color to separate, rank, or connect information.
- Watch for high-chroma colors causing visual halo or fatigue.
- Warm/cool colors affect perceived depth; use them deliberately.
- Desaturate or quiet secondary elements when distance or hierarchy is needed.

Source card: `A4A10A`.

## Image And Text

- Put text in low-information image areas.
- Avoid placing text over high-contrast, high-detail, or color-chaotic regions.
- Add overlays only when they improve legibility without hiding the subject.

Source card: `A4E4E1A2`.
