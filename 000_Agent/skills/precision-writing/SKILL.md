# Precision Writing

運用讀者優先原則，將冗長、模糊或空泛的文字，改寫為精準、有畫面感且具備行動導向的高質感內容。

## 使用原則

- **讀者優先 (Reader First)**：寫作的本質是降低讀者的認知負荷，節省讀者的寶貴時間。
- **ROA 框架**：在動筆與修改前，必須釐清：
  - **Reader (讀者)**：讀者是誰？他們目前理解到哪？他們的閱讀阻力（時間有限、沒興趣、看不懂）是什麼？
  - **Objective (目標)**：希望這篇文字在讀者腦中建立什麼認知、改變什麼態度？
  - **Action (行動)**：讀完後，希望讀者採取什麼具體下一步？
- **名詞與動詞優先**：減少空泛的形容詞與副詞（如「非常棒的」、「迅速地」），改用具體名詞與強動詞（如「省下 3 小時」、「拆解了」）直接呈現事實與動作。
- **結論先行 (Pyramid Principle)**：段落與文章結構以金字塔原理建構，先給出結論，再給支持論點與細節，讓忙碌的讀者能快速抓到重點。

## 快速流程

1. **診斷現有文字 (Diagnostic)**：
   - 抓出冗餘字詞（如「進行一個...的動作」、「關於...的部分」）。
   - 找出模糊概念，將其具體化。
   - 評估段落焦點是否單一，結構是否清晰。
2. **定位 ROA 核心 (Core Mapping)**：
   - 明確本段文字的讀者、目的與期待行動。
3. **進行精準改寫 (Refining)**：
   - **精簡 (Conciseness)**：刪除至少 20-30% 的無效字，合併重複概念。
   - **畫面感 (Imagery)**：運用場景、互動、特寫三要素，將抽象描述轉為具體畫面。
   - **字詞替換 (Word Choice)**：以「主詞 + 強動詞 + 具體名詞」重構句子。
4. **設計吸引人的標題 (Hook Design)**：
   - 若有需要，設計結合讀者痛點、具體數據、前後對比或好奇落差的標題。
5. **輸出對比與改寫說明 (Explanation)**：
   - 提供「原版 vs. 改寫版」對比，並點出 1-2 個關鍵修改點的理由，讓使用者學習寫作心法。

## 精準寫作核心技法

### 1. 畫面感三要素 (Visual Imagery)
當使用者需要將文字寫得「生動」、「有張力」時，使用以下要素重構：
- **場景 (Scene)**：設定具體的時間、地點與環境背景，讓讀者瞬間進入情境。
- **互動 (Interaction)**：寫出人與人、或人與物之間的具體互動過程。
- **特寫 (Close-up)**：捕捉感官細節（聽覺、視覺、觸覺）與具體數字，避免用概括性字眼。
- *範例*：
  - *抽象*：「他工作非常認真，經常熬夜。」
  - *畫面*：「凌晨兩點，台北辦公室只剩他那一格日光燈亮著，桌上堆了三罐喝光的黑咖啡空罐，他的鍵盤敲擊聲在空曠的辦公室裡格外清晰。」

### 2. 精煉文字檢查清單
- **消滅贅字**：
  - 刪除「進行...的動作」、「做一個...的調整」、「關於...的部分」、「...的狀況」。
  - 刪除無意義修飾詞，如「其實」、「基本上」、「基本上來說」、「非常」。
- **動詞與名詞化**：
  - 將「我們對此問題進行了深入的討論與分析」改為「我們深入分析了這個問題」。
- **主動語態優先**：
  - 避免被動語態，除非主詞不明確。

### 3. 標題與鉤子設計技法
- **具體數字**：用「省下 80% 時間」代替「大幅提升效率」。
- **好奇落差 (Curiosity Gap)**：指出讀者以為正確但實際上錯誤的事（例：「為什麼每天努力寫卡片，你的知識系統卻依然一片混亂？」）。
- **痛點直擊**：直接描繪讀者半夜睡不著、或者在工作中最痛苦的那個瞬間。

## 與其他 skill 的邊界

- `article-enrichment`：本 skill 專注在**文字精煉與寫作手藝本身**；`article-enrichment` 則是調用 `study` / `body` 卡片盒的**外部知識素材與方法論來擴充文章內容**。
- `ak-threads-booster`：本 skill 專注於通用寫作技術；`ak-threads-booster` 則是針對 Threads 平台的特有受眾、起草格式與發文節奏進行優化。
- `problem-framing`：當文字混亂是因為「底層想法還沒理清楚」時，先引導使用者使用 `problem-framing` 釐清本質，再用本 skill 精煉文字。
