# Self Brain Capture Workflow

## 目標

把對話、想法、決策、人物/專案脈絡整理進 `400_Knowledge/self_brain`，讓不同 AI 都能讀取同一份長期自我分身資料。

## 使用時機

使用者說：

- 記住這件事
- 沉澱到自我分身
- 整理成長期記憶
- 幫我建立個人腦
- 這是我的偏好 / 原則 / 決策
- 這個人 / 專案 / 關係之後要追蹤
- 把這段對話整理進知識庫

## 執行方式

本 workflow 只是快捷入口；正式 SOP 只維護在：

- `000_Agent/skills/self-brain/SKILL.md`

觸發此 workflow 時，先讀取並遵守 `self-brain` skill，再依使用者明確指令執行。
