# Project Memory Workflow

## 目標

讓 AI 在新專案、長任務、跨工具交接時，有一組可重讀的專案脈絡文件。

## 執行方式

本 workflow 只是快捷入口；正式流程只維護在：

- `000_Agent/skills/project-memory/SKILL.md`

常用口令：

```text
讀完 memory-bank，接著只做 implementation-plan 的下一步；完成後更新 progress.md 和 architecture.md。
```

適用於 app、工具、自動化、技術研究實作；不適用於日記、個人偏好或外部原始素材整理。
