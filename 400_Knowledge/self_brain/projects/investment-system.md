# Investment System

## Summary

使用者看待投資的重點是建立可重複、可驗證、可停損的系統，而不是單次博弈。核心模式包含順勢交易、籌碼觀察、模擬驗證、分批停利與固定比例風險控管。

## Current Understanding

- 投資觀偏向從失敗中學習，建立長期會獲利的系統。
- 曾從技術分析、波浪理論等工具，逐步納入籌碼面、大戶集中度、分點買賣與市場心理觀察。
- 對「抄底」心態保持警惕，較偏向亞當理論式的順勢交易。
- 對高風險資產採小額「買樂透」定位，不把它們當核心資產。
- 在真金白銀投入前，偏好先用模擬戶或小額測試驗證策略。
