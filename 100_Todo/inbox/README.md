# 100_Todo/inbox

這裡放尚未分類、尚未決定是否採用、或只是本輪討論用的臨時檔案。

## 放這裡的內容

- 外部建議或待查核片段。
- 像 `Question.md` 這類討論草稿。
- 還不知道該放 rules、skills、templates、workflows、reference 或 knowledge 的素材。
- 本輪整理中暫時產生、但不應放在 repo root 的文件。

## 不放這裡的內容

- 已確認的長期規則：放 `000_Agent/CORE_RULES.md`。
- 可重複執行 SOP：放 `000_Agent/skills/<skill>/SKILL.md`。
- 固定流程入口：放 `000_Agent/workflows/`。
- 外部原始素材：放 `200_Reference/imported/`。
- 長期個人脈絡與決策：放 `400_Knowledge/self_brain/`。
- 每日紀錄與 session log：放 `300_Journal/YYYY-MM/YYYY-MM-DD.md`。

## 清理規則

每次整理時，把 inbox 內容判斷成三類：

1. 採用：移到正式位置。
2. 保留參考：移到 `200_Reference/imported/` 或對應 reference。
3. 不採用：移到 `100_Todo/archive/`，並簡短記錄原因。
