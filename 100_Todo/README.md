# 100_Todo

此目錄用於存放**進行中、暫存與待辦**性質的內容。它是一個流動區域，任務完成或資料分類後應移出此處。

## 目錄結構與用途

- **`inbox/`**：臨時收集桶。尚未分類的臨時討論、外部建議、隨手筆記或待判斷素材先放這裡。整理後應轉移到知識庫 (`400_Knowledge`)、參考區 (`200_Reference`) 或正式封存。
- **`projects/`**：進行中的專案工作區（例如程式開發專案、大型企劃草稿）。專案結束後，產出應歸檔或轉移。
- **`drafts/`**：仍在撰寫、還沒定稿的文章或文件草稿。

## 邊界規則

- **這裡不是長期保存區**：請定期清理 Inbox 與 Drafts，不要把長期知識或重要決策留在這裡。
- **與 `self_brain/inbox` 的差異**：此處的 `inbox` 是專案與任務等級的暫存；而 `400_Knowledge/self_brain/inbox` 是確定屬於「個人長期脈絡」但尚未決定分類的內容，兩者意義不同。
