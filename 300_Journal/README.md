# 300_Journal

此目錄用於存放**以時間為序列**的紀錄檔。包含每日流水帳、工作復盤、AI 協作的 Session Log 等。

## 目錄結構與用途

- 採用 `YYYY-MM/YYYY-MM-DD.md` 的格式歸檔。例如：`2026-06/2026-06-06.md`。
- **紀錄內容**：每日目標、遭遇的問題、與 AI 協作的關鍵對話紀錄或除錯過程。

## 邊界規則

- **不當作長期真相來源**：日誌是歷史軌跡，可以用來提供素材，但如果有決定性的長期決策、偏好改變或專案狀態變更，必須將結果沉澱至 `400_Knowledge/self_brain`。
- 新的日誌統一寫入此處，不再使用舊版的 `000_Agent/memory/daily/`。
