# README_LOCAL

## 原本用途

`pro-kit` 是來源 repo 的進階教學/模板文件集合，內容包含 agent folder setup、skill creator bootstrap、tool integration、brainstorm、social cards、landing page、cross-device sync 等主題。

## 我可以怎麼用

- 當作本地 AI-Agent 架構、skills 設計與工作流整理的參考文件。
- 優先閱讀與本次匯入 skill 對應的 `05-social-cards.md`、`06-landing-page.md`。
- 涉及 hooks、MCP、外部工具或同步設定的章節，先當文件參考，不直接執行。

## 是否適合改成 SOC / GAS / fullstack 工作流

適合拆成三類本地工作流：SOC 值班/回顧模板、GAS 工具開發流程、fullstack 需求到交付流程。建議只抽取方法與文件結構，不直接套用會改全域設定的命令。
