# 200_Reference

此目錄用於存放**外部導入、靜態參考資源與樣板**。這裡的內容是作為「素材」使用，而不是執行的「規則」。

## 目錄結構與用途

- **`imported/`**：外部匯入的資料、他人的 skill/pro-kit、或其他平台的鏡像備份（如舊版知識庫）。
- **`writing-samples/`**：貼文、文章、作品的「原始」純淨樣本，用來作為風格參考，或是供 AI 分析寫作語氣之用。
- **`templates/`**：範本、固定格式、可重複套用的版型（例如特定的 Prompt 框架或文件樣板）。

## 邊界規則

- **不當作正式規則**：不要把 `imported/` 裡的外部設定或工作流當作當前 AI-Agent 的正式執行規則。正式規則只維護在 `000_Agent/CORE_RULES.md` 和 `000_Agent/skills/` 中。
- **寫作聲音分析**：原始寫作樣本放在本區；但分析出來的「寫作人格/語氣指南」，應存放在 `400_Knowledge/self_brain/writing_voice/`。
